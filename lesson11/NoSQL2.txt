Для поиска электронного адреса по имени пользователя создается пара ключ-значение
  > SET <name> <email>
где <name> - имя пользователя, <email> - адрес пользователя
Получить адрес можно командой
  > GET <name>

Для поиска имени пользователя по его электронному адресу создается пара ключ-значение
  > SET <email> <name>
где <email> - адрес пользователя, <name> - имя пользователя
Получить адрес можно командой
  > GET <email>
